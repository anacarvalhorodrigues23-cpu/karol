# karol — Site simples do Blog do Agrinho

Este repositório agora contém uma versão inicial de um site sóbrio para o "Blog do Agrinho".

Arquivos adicionados:

- `index.html` — página principal do blog (layout sóbrio e legível).
- `styles.css` — estilos básicos para o site.

Como usar

1. Abra `index.html` no navegador (duplo clique ou `file://`), ou coloque os arquivos em um servidor estático.
2. Para publicar no GitHub Pages, ative GitHub Pages nas configurações do repositório e aponte para a branch `main` e a pasta `/`.

Próximos passos sugeridos

- Adicionar posts em formato Markdown e um gerador simples (ex: Eleventy, Hugo) se quiser automatizar páginas.
- Criar um layout de post único (`post.html`) e páginas de categoria.
- Substituir o e-mail de contato por um real ou formulário.

Contribuições são bem-vindas — abra uma issue ou um pull request com melhorias.
